### 홈화면

- 바로 다음에 울릴 알람이 무엇인지 보여주는 기능
- 알람
  - 알람 이름, 시간, 날짜 출력
  - 반복 횟수 출력
  - 알람 on/off 기능
  - 알람 순서 바꾸는 기능
---
<img src="https://velog.velcdn.com/images/new/post/c1090eee-cfd5-4cd4-ae80-6505c602f9f1/image.jpg" width="300px">

### 알람 추가/수정 화면

- CupertinoDatePicker를 이용한 시간 설정
- 반복 알람일 경우 다음 알람 날짜로 시작일을 설정할 수 있는 기능 (예시: 시작일 11월 3일에 매주 토요일에 반복하는 알람일 경우 > 버튼을 누르면 시작일이 11월 10일이 되는 기능)
- 간편하게 반복 요일을 설정할 수 있는 요일 버튼
- 알람음, 진동, 반복 주기를 출력하고 on/off 할 수 있음

### 날짜 선택 화면

- 지정한 날짜로 알람이 울리는 날짜를 선택할 수 있음
  - 반복 알람이 아닌 일회성 알람이 됨

### 반복 날짜 선택 화면

- 일/주/월/년 중 원하는 주기로 알람을 반복할 수 있는 기능
- 시작일과 종료일 설정 가능
- 휴일 선택 기능으로 알람이 안 울리는 날짜 설정 가능

### 더보기 화면

- 각종 설정을 할 수 있는 화면
  - 알람 안정화시키기: 안드로이드 환경에서 알람이 잘 작동할 수 있게 권한을 설정하는 기능
  - 첫 화면 폴더: 앱을 실행시킬 때 첫 화면으로 출력시킬 폴더를 정하는 기능
  - 정렬: 설정순/날짜순으로 알람을 정렬시킬 수 있는 기능
  - 언어: 한글/English로 언어를 설정할 수 있는 기능
  - 테마: Green/Dark 테마로 테마를 변경할 수 있는 기능
  - 초기화: 알람 앱의 데이터를 초기화하는 기능
